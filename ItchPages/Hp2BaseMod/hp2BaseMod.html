Hp2BaseMod

A modding framework for HuniePop 2 - Double Date.

On it's own this mod doesn't affect gameplay, but it does provide handling and utility for other mods! It is highly recommended you also install Hp2BaseModTweaks, which provides modded UI support and extra options.

Installation

The Hp2BaseMod is built on BepInEx. You must install BepInEx to run it or any dependant mods. Click the link or see the tutorial for more information.

Once BepInEx is installed, unzip and place the Hp2BaseMod folder in
HuniePop 2 - Double Date\BepInEx\plugins

Other mods based on the Hp2BaseMod must also be placed in
HuniePop 2 - Double Date\BepInEx\plugins
